# Grocery Store

A simple grocery store web app built with Angular (frontend) and JSON Server (backend).

## Run the project

1. Start backend
    - Go to `backend`
    - Run `json-server --watch db.json`

2. Start frontend
    - Go to `frontend`
    - Run `npm install`
    - Run `npm start`

Frontend runs on http://localhost:4200 and backend runs on http://localhost:3000.
